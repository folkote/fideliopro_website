Модуль Завтраки с планшета

С помощью модуля Завтраки с планшета можно быстро посмотреть какие пакеты услуг должны быть оказаны сегодня. Посмотреть список гостей по одному или нескольким пакетам и отметить, что услуга была оказана.

  • Работает на любых планшетах
  • Интуитивный интерфейс
  • Учет израсходованных и оставшихся пакетов услуг
  • При желании, можно сохранять замеры температуры

Модуль Профиль Брони

Модуль "Профиль Брони" представляет собой страничку на сайте отеля. Гость получает её в письме подтверждения и перейдя по ней может

  • Внести предоплату по брони, либо полностью ее оплатить
  • Загрузить изображение паспорта для предварительной регистрации
  • Заказать дополнительные услуги
  • Указать свои предпочтения по размещению
